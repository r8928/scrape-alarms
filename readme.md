# scrape-alarms

Scrape Alarms.com

### Installation

scrape-alarms requires [Node.js](https://nodejs.org/) to run.

Use [pnpm](https://pnpm.js.org/en/installation) package manager.

```sh
choco install nodejs -y
npm install -g pnpm
pnpm install
```

### Run for all locations
```sh
node index.js
```

### Run for sepecific location
```sh
node index.js 664e
```
